# Bienvenido a tu Expo app👋

# Instalar Expo App en tu celular

1. En tu instalador de aplicaciones, buscar la aplicación "Expo App" e instalarla.

2. Una vez instalado, ver la version del bun con el comando
   bun --version

# Clonar el Repositorio

1. Entrar al siguiente enlace: https://github.com/luciaacunaa/GRITA-DDS.git

2. Copiar el enlace y en tu terminal usar los siguientes comandos:

git clone https://github.com/luciaacunaa/GRITA-DDS.git

cd lista-compras

# Instalar Bun en tu terminal

1. Ejecutar el siguiente comando:

bun install

# Escanear QR

1. Ya hecho todos los pasos, utilizar el siguiente comando:

bun start

# Alternativas:

# bunx expo start

# npm run start / npx expo start

2. Se abrirá Expo DevTools en el que verás un Código QR

3. En tu celular, escanear el código QR que aparece en la terminal.

4. Esperar unos segundos y listo ¡Disfruta de la App!
